I'm Mus, a final year PhD student in Physics at Cambridge University. You can see some of the things I like, enjoy or like to think about. I've tried to juggle most of these but, more so recently, the PhD and experiments have taken over... I work with Graphene which is a 2010 Nobel prize winning atomically thin material. I use graphene to filter ions from salts in the hope of desalinating water and using the ion flow to generate clean power. Studying this ion flow reveals interesting physical phenomena which I model using a theoretical framework. Recently, I have created nano-holes in graphene in order to pass DNA through and sense their passage - with the aim of decoding information. Increasingly now, I'm trying to use signal analysis and machine learning techniques to better understand these phenomena....
